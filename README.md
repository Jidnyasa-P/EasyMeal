🍽️ Cafeteria / Mess Menu & Order Portal (C-MOP)
A web-based platform designed to streamline cafeteria operations by allowing students to view menus, place food orders, and track their orders in real time while enabling cafeteria administrators to manage menus and orders efficiently.

📌 Project Overview
The Cafeteria / Mess Menu & Order Portal (C-MOP) is designed to simplify and digitize the traditional cafeteria management system.
Instead of relying on manual ordering methods, this system provides an online interface where students can browse available food items, place orders, and track order status. At the same time, cafeteria staff can manage menus and monitor incoming orders efficiently.
The platform improves transparency, reduces waiting time, and enhances communication between students and cafeteria staff.

🎯 Objectives
Provide an easy way for students to view cafeteria menus
Enable online food ordering
Allow students to track order status
Help cafeteria staff manage menu items and orders
Improve cafeteria workflow and reduce manual errors

👥 User Roles
1. Student / User
Students using the system can:
Register and log in to the portal
View the cafeteria menu
Place food orders
Track their order status
2. Admin / Cafeteria Staff
Administrators can:
Manage menu items
View incoming orders
Update order status
Maintain the cafeteria system

⚙️ Core Features
🔐 User Authentication
User registration
Secure login system
Role-based access (User / Admin)
📋 Menu Viewing
Display available food items
Show item details such as name and availability
🛒 Food Ordering
Users can select items and place orders
Orders are recorded in the system
📦 Order Tracking
Users can view order status
Real-time order updates
🛠 Admin Management
Add, update, or remove menu items
View all orders placed by users
Update order status

Server-side backend for data processing
